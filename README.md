## 项目简介

**xuanniaohuishi (玄鸟绘世)** — MCP 运动规划无人机喷涂绘画系统。

基于 ROS 的自主无人机喷涂平台，核心流程为 **AI 生成设计 → 轨迹提取 → 采样点生成 → MCP 运动规划 → 无人机喷涂执行**。

无人机像一支"空中画笔"，在幕布/墙面等载体上精准喷涂 AI 生成的图案。

### 核心工作流

```
┌──────────┐    ┌────────────┐    ┌──────────┐    ┌──────────┐    ┌──────────┐
│ AI 生图   │───→│ 轮廓提取    │───→│ 轨迹采样   │───→│ MCP 规划  │───→│ 无人机喷涂 │
│          │    │ Canny+      │    │ 连续采样点 │    │ 路径优化  │    │          │
│ Stable   │    │ findContours│    │ 密度可调   │    │ TSP+样条  │    │ 执行喷涂  │
│ Diffusion│    │            │    │           │    │ 插值平滑  │    │          │
└──────────┘    └────────────┘    └──────────┘    └──────────┘    └──────────┘
       ↑                                                              │
       └────────────────── 视觉反馈闭环 ───────────────────────────────┘
                          (覆盖率/质量检测)
```

### 技术栈

- **框架**: ROS (Melodic/Noetic)
- **AI 生图**: Stable Diffusion / ControlNet 管线
- **轨迹规划**: OpenCV 轮廓提取 + TSP 路径优化 + 样条插值
- **视觉反馈**: YOLOv8 + OpenCV 喷涂覆盖率检测
- **UI**: PyQt5 地面站
- **坐标系**: 幕布物理坐标系（可配置尺寸）

### MCP 轨迹规划器

核心模块 `mcp_planner.py` 提供：

| 功能 | 方法 | 说明 |
|------|------|------|
| 设计加载 | `load_design(path)` | 加载 AI 生成图（PNG/SVG/BMP） |
| 轮廓提取 | `extract_trajectory()` | Canny 边缘检测 + findContours 笔画提取 |
| 采样生成 | — | 沿轮廓路径等距或自适应采样 |
| 路径优化 | `optimize_path()` | TSP 求解最优笔画顺序，减少空驶 |
| 轨迹映射 | `generate_waypoints()` | 图像坐标 → 无人机物理坐标 |
| 轨迹平滑 | `smooth_trajectory()` | 三次样条/贝塞尔插值平滑 |
| 可视化 | `visualize_path()` | OpenCV 窗口预览规划路径 |

### 快速开始

```bash
# 依赖安装
pip install torch ultralytics opencv-python pyyaml scipy
sudo apt install ros-${ROS_DISTRO}-mavros ros-${ROS_DISTRO}-cv-bridge

# 启动 MCP 喷涂地面站
roslaunch xuanniaohuishi mcp_painter.launch
```

### 使用流程

1. **AI 生图**: 使用 Stable Diffusion 或其他 AI 工具生成设计图案
2. **轨迹提取**: 启动地面站，加载设计图，自动提取喷涂轨迹
3. **轨迹预览**: 预览规划的路径，调整采样密度/喷涂参数
4. **开始喷涂**: 无人机按规划轨迹执行喷涂
5. **质量检测**: 视觉系统实时反馈喷涂覆盖率
